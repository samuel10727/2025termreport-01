Tech Stack Introduction

地球物理雲端工具箱

在現代地球物理學中，數據的展示、管理與記錄與理論同等重要。本網頁將介紹三項核心數位工具。

GitHub & Codespace: 程式碼的家

為什麼地球物理需要它？

GitHub 是全球最大的版本控制平台。透過 Codespace，我們可以直接在雲端瀏覽器撰寫 ObsPy 或 PyGMT 程式，無需在個人電腦安裝複雜的環境。

Step 1: 建立 Repository

將課程所有的 .ipynb 或 .html 檔案集中管理。

Step 2: 啟動 Codespace

點擊 "Code" > "Codespaces" 即可開啟雲端 VS Code 環境。

Hugging Face Spaces: 互動展示平台

將數據轉化為應用 (Web App)

Hugging Face 不只是 AI 模型的集散地，其 Spaces 功能讓地球物理數據能以 Gradio 或 Streamlit 呈現互動介面，讓非專業人士也能點擊分析震測資料。

  • 支持 Python 為主導的互動開發
  • 免費部署，且具備強大的社群開源資源

Google Blogger: 學習日誌與反思

科學家為什麼要寫部落格？

記錄是科學研究的基石。透過 Google Blogger，我們可以系統化地整理每一週的野外筆記、推導過程以及與專家講座的心得。

教學重點： 建議每篇文章包含「理論目標」、「實作截圖」與「錯誤修正 (Debug)」三個部分。

心得與反思

作為地球物理家，程式碼就是我們的實驗器材。掌握 GitHub 能讓你與全球研究者協作，Hugging Face 能展現你的數據說服力，而 Blogger 則是你的數位大腦。希望各位同學能善用這些工具，建立起屬於自己的學術品牌。
