二、科學思路 (Scientific Reasoning)

重力修正的目的是將觀測到的重力值 ($g_{obs}$)，扣除掉「已知地理因素」的影響，剩下的差異即為地底密度不均勻造成的異常 (Anomaly)。

1. 理論重力 (Theoretical Gravity)

考慮地球自轉與橢球體形狀。隨緯度增加，重力值會上升。使用 1967 國際重力公式。

2. 空間修正 (Free-Air Correction, FAC)

隨著測站離地心越遠（高度增加），引力會減弱。FAC 補償了這部分損失。

3. 布蓋修正 (Bouguer Correction, BC)

補償測站與海平面之間多出來的岩層質量引力（假設密度 $\rho = 2.67 g/cm^3$）。

三、數值計算結果

# 計算步驟摘要

1. Theoretical Gravity ($g_{th}$):
    $g_{th} = 978031.85 (1 + 0.00527889 \sin^2 \phi + 0.00002346 \sin^4 \phi)$
    ≈ 980,884.28 mGal

2. Free-Air Correction (FAC):
    $FAC = 0.3086 \times h = 0.3086 \times 487.9$
    ≈ 150.57 mGal

3. Bouguer Correction (BC):
    $BC = 0.04193 \times \rho \times h = 0.04193 \times 2.67 \times 487.9$
    ≈ 54.62 mGal

4. Free-Air Anomaly (FAA):

    $FAA = g_{obs} - g_{th} + FAC = 980717.39 - 980884.28 + 150.57$
    FAA = -16.32 mGal

5. Bouguer Anomaly (BA):

    $BA = FAA - BC = -16.32 - 54.62$
    BA = -70.94 mGal

(b) 題思考：海洋環境

若位於海洋表面（水深 487.9m），空間修正為 0（因位於基準面）。布蓋修正則需考慮「水代替岩石」的質量損失。此處 BA 修正值會改為密度差修正：$0.04193 \times (2.67 - 1.03) \times depth$。

學習反思 Reflection

這道題目展示了布蓋異常在實際地質解釋中的關鍵性。雖然 FAA 反映了最直接的剩餘引力，但 BA 扣除了局部高程的質量影響，更能反映出地殼深部的結構特徵（如莫氏面深度或隱藏的地殼密度體）。在海洋環境下，計算邏輯的轉變也提醒我們，地球物理模型的建立必須依賴於正確的環境背景假設。
